## Habilidades Disponíveis

- **brainstorm** - Guia para brainstorming de ideias e transformá-las em designs totalmente formados através de questionamento iterativo
- **code-review** - Guia para realizar revisões de código usando agentes competidores para encontrar problemas de arquitetura e implementação
- **decisions** - Guia para registrar decisões arquiteturais e de design significativas em `docs/decisions.md`
- **git** - Guia para usar git de acordo com preferências (nomenclatura de branches, mensagens de commit, referências a issues)
- **go** - Guia para desenvolver apps Go e módulos/bibliotecas (estilo de código, testes, injeção de dependência, estrutura de pacotes)
- **gomponents** - Guia para trabalhar com gomponents, uma biblioteca de componentes HTML em Go puro para construir views HTML
- **journal** - Guia para usar o banco de dados de diário persistente da IA para capturar insights e lições aprendidas
- **nanobanana** - Guia para gerar e editar imagens usando IA generativa com a CLI nanobanana
- **save-web-page** - Guia para salvar uma página web para uso offline usando a [CLI monolith](https://github.com/Y2Z/monolith)
- **skill-creator** - Guia para criar habilidades eficazes que estendem as capacidades da IA com conhecimento especializado e fluxos de trabalho
- **worktrees** - Guia para usar git worktrees para paralelizar o desenvolvimento com agentes de codificação em ambientes isolados
