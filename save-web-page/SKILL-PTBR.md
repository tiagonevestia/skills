# Salvar página web

Você pode usar a ferramenta de linha de comando `monolith` para salvar uma página web para uso offline.

O comando principal a ser usado é `monolith --isolate -o <nome>.html <url>`.

Prefira salvar no diretório `~/AI/web/`, a menos que instruído de outra forma.

Se precisar fazer qualquer outra coisa com o comando, use `monolith -h` para ver seus usos.
